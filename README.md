# Financial Market & Macroeconomic Dashboard

This project is a Power BI dashboard that analyzes stock market performance alongside key macroeconomic indicators.

## Objective
To explore how major stock indices behave under different economic conditions using combined financial and macroeconomic data.

## Key KPIs
- Average Close Price
- Total Trading Volume
- Average Inflation Rate
- Average Interest Rate
- Average GDP Growth

## Dashboard Features
- Market trend analysis over time
- Trading volume comparison by year
- Inflation vs daily return scatter analysis
- Interactive slicers for index and year filtering

## Tools Used
- Power BI
- DAX
- Data Cleaning
- Data Modeling

## Files in this Repository
- `.pbix` file: Power BI dashboard
- `.csv` file: Dataset used
